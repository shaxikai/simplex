# Simplex
------------------------------------------------------------
## Brief Introduction
Linear Programming Simplex Method 

## Usage
If you are using linux systems.
Obtain the sample sequence and launch:
Input in the terminal
```
| mkdir build
| cd build
| cmake ..
| make 
| ./Simplex
```
